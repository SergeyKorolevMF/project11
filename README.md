# Meeting Manager Bot (Vibe Tracker)

Telegram-бот для менеджеров, помогающий вести заметки со встреч (1-1, командные синки), отслеживать настроение сотрудников и автоматически конвертировать записи в задачи.

## Возможности (MVP)
*   **Ведение заметок:** Свободный ввод текста, который сохраняется с привязкой к дате.
*   **Профили сотрудников:** Создание карточек для репортов (Direct Reports).
*   **Трекинг настроения:** Опциональные поля "Уровень стресса", "Крыса недели", "Акула недели".
*   **База данных:** История встреч хранится в SQLite.

## Технологии
*   Python 3.11+
*   [aiogram 3.x](https://github.com/aiogram/aiogram) - Асинхронный фреймворк для Telegram Bot API
*   [Tortoise ORM](https://github.com/tortoise/tortoise-orm) - Асинхронная ORM
*   SQLite - База данных

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <repository_url>
    cd <repository_name>
    ```

2.  **Создайте виртуальное окружение:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate  # На macOS/Linux
    # venv\Scripts\activate   # На Windows
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Настройте переменные окружения:**
    *   Скопируйте пример конфига: `cp .env.example .env`
    *   Откройте `.env` и вставьте ваш токен бота: `BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`

5.  **Запустите бота:**
    ```bash
    python bot.py
    ```
